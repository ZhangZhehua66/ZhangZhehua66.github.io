## 实现部分怎么编呢？

Jia的论文是这样写的：

1. 原始的是啥样 -> 有问题 -> 为了解决 -> 笼统概括 -> 图
2. 如何x是本方案的点 -> 怎么样做不好 -> 因此可以怎么做好
3. 考虑到 -> 为了避免 -> 怎么做
4. 具体实现 -> 对使用到的东西说明
5. 工作流程，5、6段
6. 对代码和数据结构描述
